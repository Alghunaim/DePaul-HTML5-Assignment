DePaul HTML5 Assignment

Welcome to the DePaul HTML5 Assignment. The goals of this assignment are to demonstrate the HTML5 and CSS3 skills you have aquired during this course.

For this assignment, we're going to build a web page and publish it onto the internet using Github's Pages feature. To complete the assignment you will be required to do the following:

  1. Create a web page using HTML5/CSS3.
  2. Publish it to Github.com.
  3. Submit the link to your web page for grading.

Grading

Before we get into the nitty gritty of things, let's review how this assignment will be graded:

Item Points
Correct use of HTML5/CSS3 elements 50
Correct use of HTML5/CSS3 APIs 30
Publishing the site to Github.com 10
Creativity 10

The Assignment

The website you will be required to create is a two page website consisting of a home page and a detail page. Each page will be required to work correctly on a desktop browser as well as a tablet and smart phone.

I encourge you to be creative. Create a site about something that interests you. Since you will be publishing this to the internet, you will be able to reference this as a portfolio piece to future employeers.

Below is a breakdown on the points awarded for each requirement.

Element Points
Using valid semantic HTML5 markup (must be able to validate according to W3C. 25
Use of at least one media type 10
Use of at least one HTML5 form type 10
Use of web fonts 10
Use of gradient, shadow, or other CSS3 feature 5
Use of CSS3 correctly 15
Works correctly on the current release of all major desktop browsers 5
Works correctly in a tablet device 5
Works correctly in a smart phone 5
Published to Github.com using their Pages functionality. 10
Total 100

Note: There are many HTML generators, both on the internet and as desktop applications. Any use of an HTML or CSS generator will result in a zero (0) for this assignment.
